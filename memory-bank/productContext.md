# productContext.md

**The Why & The UX**
* **The Problem:** Modern AI coding tools (Mythos) are imaginatively rich but structurally unreliable (Logos), often hallucinating syntax or breaking imports.
* **The Solution:** Athanor acts as a "Legislator of Intent." The AI issues abstract **Dialectic Actions**, and the native Python engine executes them with surgical, AST-safe precision.
* **Experience Goals:** A full-screen, adaptive terminal interface (TUI) that feels like a professional IDE/IRC client, providing absolute visibility into logs, AI thought processes, and costs.
